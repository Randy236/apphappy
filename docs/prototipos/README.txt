Prototipos Happy Jump
=====================

Coloca aqui las capturas de pantalla (PNG) o un PDF con las pantallas principales:
- Login
- Pantalla principal / navegacion
- Modulo Cancha (grilla y/o calendario)
- Modulo Salones
- Admin / reportes (si aplica)
- Perfil

Puedes generarlas desde el emulador de Android Studio o desde un telefono con la app instalada.

Si usas Figma u otra herramienta, agrega un archivo ENLACES.txt con la URL del prototipo.
